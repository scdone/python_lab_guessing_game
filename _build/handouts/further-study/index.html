Guessing Game: Further Study

Let’s make our game more interesting.

Let Users Play Multiple Rounds and Track Best Scores

When the user guesses the correct number, ask them if they’d like to play again and restart the game, rather than exiting after a correct guess.

Keep track of the best score — the lowest number of guesses a user has made — and display that when you congratulate the user.

Limit the Number of Guesses a User Gets

Give the user a maximum number of opportunities to guess, like hangman. When the user reaches the maximum number, end the game with a message to the user saying “Too many tries!” and ask if they’d like to play again.

Add a Range Option

Give the user an opportunity to set the range of numbers from a start number to an end number.

Give the Computer a Turn

Add a playing option where the computer tries to guess your number. The computer should be given the start and end of the range. The computer will print a guess and the user should respond “too high”, “too low” or “you won”; Repeat until the computer gets it.
